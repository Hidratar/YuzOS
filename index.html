Transforme a Nova Guia em uma Central de Produtividade

Yuz OS é uma extensão para o navegador que substitui a nova guia por um painel poderoso, inteligente e altamente personalizável. Gerencie tarefas, acompanhe notícias de segurança e mantenha acessos rápidos sempre ao alcance.

Recursos principais

Painéis dinâmicos

Organize widgets de notícias, lista de tarefas, atalhos e notas em um layout flexível. Arraste, redimensione e centralize como preferir.

Personalização visual

Escolha temas claros ou escuros, selecione planos de fundo e defina cores e fontes. O motor de temas ajusta automaticamente a legibilidade.

Lista de tarefas integrada

Adicione tarefas rapidamente e acompanhe seu progresso. As listas ficam sempre sincronizadas com o resto do painel.

Atalhos rápidos

Acesse ferramentas como Nmap, Wireshark e Metasploit, ou adicione seus próprios aplicativos ao menu lateral com um clique.

Controle total

Edite widgets existentes, crie novos a partir de URLs, scripts ou texto, e exclua elementos que não precisa. Tudo sem complicações.

Documentação

Instalação e uso

Para instalar Yuz OS, basta adicionar a extensão ao seu navegador via Chrome Web Store. Após a instalação, abra uma nova guia e aproveite o painel totalmente renovado. Os dados são armazenados localmente, garantindo privacidade e desempenho.

Widgets e ferramentas

Cada widget pode ser movido, redimensionado e centralizado. Pressione Tab enquanto arrasta para alinhar perfeitamente com outros elementos. Os botões no canto de cada widget permitem expandir, reduzir ou remover conforme sua necessidade.

Personalização avançada

O menu de configurações oferece controles precisos sobre temas, cores e fontes. Ao escolher "Criar/Editar Tema", você pode definir cores primárias e secundárias que combinam com seu estilo. O motor de temas detecta o brilho do plano de fundo para ajustar automaticamente o contraste e melhorar a legibilidade.

function saveConfig(config) {
  localStorage.setItem('yuzOcsConfig', JSON.stringify(config));
  applyConfig(config);
}

Acima está um exemplo simples de como as configurações são salvas no armazenamento local e aplicadas imediatamente. A extensão trabalha de forma transparente para que você foque no que importa.

Desenvolvedores

Yuz OS possui suporte a widgets personalizados. É possível integrar APIs ou exibir dados remotos com JavaScript padrão e fetch utilizando o arquivo autowidgetconget.js. Os widgets podem exibir texto simples, conteúdo de uma URL ou executar código customizado.
